注册与验证

本地测试环境可以直接在这里关闭邮箱验证码和人机验证。

填写后，登录页和注册页会显示 Google 登录按钮，并使用这个 Client ID 做令牌校验。

先保存 Apple Client ID 以便统一管理。前端 Apple 登录流程尚未接入，当前只保留配置入口。

关闭邮箱验证码后，注册页会直接放行邮箱和验证码步骤；关闭人机验证后，不再加载所选的人机验证组件。Google Client ID 保存在这里后会立即影响公开登录页。

上传与预览

上传与预览

把上传限制和预览策略分成两个独立配置区，便于控制文件接入边界和在线查看范围。

上传策略

按需调整服务层上传大小限制和扩展名白名单。

留空表示允许全部扩展名，支持逗号或换行分隔。

预览策略

按文件族控制预览能力，并支持用扩展名进一步收窄范围。

留空表示按上方文件族开关放行；填写后会额外按扩展名收窄。
